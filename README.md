📊 Student Performance & Grit Analysis Dashboard

This project explores the relationship between student academic performance and grit (perseverance and passion for long-term goals).
The dashboard was developed using Power BI to transform raw educational data into meaningful and interactive insights.

🎯 Objective:
To understand how grit impacts student performance and identify patterns that contribute to academic success.

💡 Insights (LinkedIn-ready)
📌 The overall academic success rate is 69%, indicating moderate performance with clear room for improvement.
📌 Behavioral factors such as grit, motivation, and attendance appear to play a critical role in pushing success beyond the average level.
📌 Academic success is not only about study hours, but also consistency 

🎯 Recommendations
🔹 Enhance Grit & Motivation Programs
Implement structured resilience-building and motivation initiatives to strengthen students’ perseverance and long-term commitment.
🔹 Reduce Absenteeism as a Core KPI
Track and manage attendance as a key performance indicator, as it has a direct impact on academic outcomes.
🔹 Support Below-Average Students with Behavioral Interventions
Develop targeted support plans that address behavioral patterns 

🛠 Tools Used:

Power BI
Data Cleaning & Transformation
DAX Measures

📈 Outcome:
The dashboard helps educators and decision-makers understand behavioral and academic trends, supporting data-driven strategies to improve student outcomes.
